Build a complete, production-ready, world-class personal portfolio website for a multidisciplinary digital product expert.

## 1. Core Vision

Create a premium futuristic portfolio experience built around the concept:

**“One Creative Brain. Complete Digital Solutions.”**

The homepage must not look like a traditional portfolio with many stacked sections. It should feel like one immersive, cinematic hero experience.

At the center of the hero, create a large interactive 3D digital brain. Animated neural connections should extend from the brain toward floating expertise cards.

The website should communicate that I can take a digital product from initial idea to branding, UI/UX design, development, AI integration, launch, and continued improvement.

The final website must feel like one of the best creative developer and designer portfolios in the world.

Do not create a generic agency template.

Typography should feel modern, premium, bold, and highly readable.

Recommended hierarchy:

* Hero headline: 72px–96px desktop
* Tablet headline: 56px–72px
* Mobile headline: 42px–52px
* Section headings: 40px–64px
* Body text: 16px–18px

Use strong font weights for headlines and softer weights for descriptions.

## 7. Interactive 3D Brain

Create a high-quality interactive digital brain using Three.js and React Three Fiber.

The brain should:

* Sit near the center of the hero
* Be built from particles, lines, wireframe geometry, or a high-quality optimized 3D model
* Have subtle animated electrical impulses
* Slowly rotate or respond to cursor movement
* React gently when users hover over expertise cards
* Send light pulses through neural connections
* Have subtle breathing or energy animation
* Look professional and technologically advanced
* Maintain good performance

Add a soft radial light behind it.

When a user moves the cursor:

* The brain should rotate slightly
* Nearby particles should respond
* Light intensity can subtly change
* Neural connection lines should feel alive

When a user hovers over an expertise card:

* The corresponding neural path should brighten
* A pulse should travel from the brain to the card
* The brain should highlight a specific region
* The selected card should move slightly closer to the viewer

When a card is clicked:

* Create a cinematic transition
* Animate the pulse from the brain to the selected card
* Expand the card
* Transition to the relevant expertise page

For mobile devices:

* Reduce particle count
* Simplify the 3D effect
* Disable expensive cursor tracking
* Preserve the premium visual experience
* Provide a static or lightweight animated fallback where necessary

## 16. Animation System

The animation quality is one of the most important parts of the website.

Create smooth, sophisticated animations inspired by award-winning digital studios.

Use animation intentionally.

### Page load animation

* Dark screen
* Small central neural spark
* Spark expands into brain particles
* Brain becomes visible
* Connection lines activate
* Expertise cards appear sequentially
* Main headline reveals with masked animation
* Navigation fades into position

Keep the intro under approximately three seconds and allow it to be skipped.

### Text animations

* Masked line reveals
* Word-by-word entrance
* Subtle blur-to-sharp effect
* Letter spacing settles naturally
* No excessive bouncing

### Card animations

* Smooth perspective tilt
* Soft border illumination
* Background depth shift
* Arrow movement
* Connection pulse
* Magnetic hover behavior
* Strong focus states for keyboard users

### Page transitions

* Selected neural connection expands
* Screen transitions through a soft light pulse
* New page content reveals smoothly
* Preserve navigation continuity

### Scroll animations

* Use restrained parallax
* Reveal content progressively
* Animate timelines as they enter the viewport
* Keep text readable at all times
* Avoid excessive scroll hijacking

### Cursor

Create a custom cursor on desktop only.

Cursor behavior:

* Small dot and outer circle
* Expands over links
* Shows “View” over project cards
* Shows “Explore” over expertise cards
* Shows “Drag” where appropriate
* Automatically disables on touch devices

### Reduced motion

Respect the user’s `prefers-reduced-motion` setting.

When enabled:

* Remove the cinematic intro
* Disable cursor effects
* Reduce 3D motion
* Replace large transitions with simple fades
* Keep all functionality available
